# Jeu du Pendu en JavaScript

## Description

Ce projet est une implémentation du célèbre **jeu du pendu** en JavaScript.  
Le but est de deviner les mots en un temps limité, en évitant de dépasser le nombre de tentatives autorisées.

---

## Règles du jeu

1. Devinez les lettres du mot en tapant les lettres sur votre clavier.  
2. Chaque lettre incorrecte réduit le nombre de tentatives restantes.  
3. Vous gagnez des points pour chaque mot correctement deviné.  
4. Attention ! Le **temps** est un facteur important : vous devrez trouver un maximum de mots en **3 minutes**.

---

## Fonctionnalités

- Sélection aléatoire des mots à deviner.  
- Gestion des lettres correctes et incorrectes.  
- Compteur de tentatives restantes.  
- Score basé sur le nombre de mots correctement devinés.  
- Timer de 3 minutes pour jouer contre la montre.  

---

## Technologies utilisées

- HTML  
- CSS  
- JavaScript (ES6)
